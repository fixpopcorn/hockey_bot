# Hockey Jersey Bot 🏒👕

**Telegram-бот для учёта хоккейных джерси в команде.**
Бот позволяет команде быстро фиксировать, кто взял какой комплект для стирки, используя username Telegram.

---

## **Особенности**

* Запись пользователя автоматически по его **username** или полному имени.
* Меню с кнопками для выбора комплекта:

  * Тренировочный
  * Тёмный игровой
  * Белый игровой
* Просмотр последних записей командой `/list`.
* Команды `/take` и `/list` отображаются в строке набора текста Telegram.
* Данные сохраняются в локальной SQLite-базе (`data/database.db`).
* Docker и Docker Compose для лёгкого развёртывания.

## **Установка и запуск локально**

1. Клонируем репозиторий:

```bash
git clone https://github.com/username/hockey_jersey_bot.git
cd hockey_bot
```

2. Устанавливаем зависимости:

```bash
python -m venv venv
source venv/bin/activate   # Linux/macOS
venv\Scripts\activate     # Windows
pip install -r requirements.txt
```

3. Создаём `.env` на основе `.env.example` и добавляем токен:

```
BOT_TOKEN=ваш_токен_бота
```

4. Создаём папку для базы данных:

```bash
mkdir -p data
```

5. Запускаем бота:

```bash
python bot.py
```

---

## **Запуск через Docker**

1. Создаём папку `data`:

```bash
mkdir -p data
```

2. Собираем и запускаем контейнер:

```bash
docker compose up -d --build
```

3. Просмотр логов:

```bash
docker compose logs -f hockey_bot
```

База SQLite (`database.db`) будет храниться в папке `data` и сохраняться между перезапусками контейнера.

---

## **Команды бота**

| Команда | Описание                                            |
| ------- | --------------------------------------------------- |
| `/take` | Выбрать комплект и записать его в базу              |
| `/list` | Показать последние записи (кто взял какой комплект) |

---

## **Пример использования**

1. В чате Telegram пишем `/take`.
2. Бот выводит кнопки с комплектами.
3. Пользователь нажимает кнопку → запись сохраняется в базу с username и временем.
4. Команда `/list` показывает последние записи:

```
2025-11-24 18:32 — user1: тренировочный
2025-11-24 18:35 — user2: тёмный игровой
```

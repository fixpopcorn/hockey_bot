# Hockey Jersey Bot 🏒👕

Telegram-бот для учёта хоккейных джерси в команде. Помогает отслеживать, кто взял какой комплект формы на стирку и у кого сейчас находится конкретный комплект.

---

## ✨ Возможности

1. `/take` — взять комплект формы.
   - Пользователь выбирает комплект.
   - Бот сохраняет информацию о пользователе и времени.

2. `/list` — показать последние записи.
   - Выводит последние 20 записей о том, кто и какой комплект взял.

3. `/holders` — узнать, у кого сейчас комплект.
   - Пользователь выбирает комплект.
   - Бот показывает последнего пользователя, который взял этот комплект.

## 🛠️ Установка и запуск

### Предварительные требования
* Установленный [Docker](https://www.docker.com/) (рекомендуется)
* Или Python 3.9+
* Токен бота от [@BotFather](https://t.me/BotFather)

### 🐳 Запуск через Docker (Рекомендуется)

Это самый простой способ запустить бота на сервере или локально.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <ссылка_на_репозиторий>
    cd <папка_проекта>
    ```

2.  **Создайте файл `.env`** в корне проекта и вставьте туда токен:
    ```env
    BOT_TOKEN=ваш_токен_от_botfather
    ```

3.  **Запустите бота:**
    ```bash
    docker compose up -d --build
    ```

4.  **Полезные команды Docker:**
    * Посмотреть логи: `docker compose logs -f hockey_bot`
    * Остановить бота: `docker compose down`
---
### 🐍 Локальный запуск (без Docker)

1.  **Создайте виртуальное окружение:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Linux/macOS
    venv\Scripts\activate     # Для Windows
    ```

2.  **Установите зависимости:**
    (Убедитесь, что у вас создан файл `requirements.txt` с содержимым: `python-telegram-bot`, `python-dotenv`, `pytz`)
    ```bash
    pip install -r requirements.txt
    ```

3.  **Настройте `.env`** (как в пункте выше).

4.  **Запустите бота:**
    ```bash
    python bot.py
    ```

## ⚙️ Настройки и Часовой пояс

> ⚠️ **Обратите внимание:**
> По умолчанию бот настроен на часовой пояс **Asia/Omsk**.

Если ваша команда находится в другом регионе, вам необходимо изменить настройку времени. Для этого откройте файл `db.py` и отредактируйте строку с timezone:

```python
# db.py
# Замените "Asia/Omsk" на ваш регион, например "Europe/Moscow"
OMS_TZ = pytz.timezone("Asia/Omsk")
```
